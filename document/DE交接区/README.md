隐私边界：本目录只引用 public-safe 的 `outputs/demo/` 汇总指标和图表，不包含真实原文、样本级预测、用户链接、原图或模型权重。

## 1. 这个项目一句话

本项目尝试用“文本内容 + 图片视觉信息 + 用户行为特征”三类信息，判断社交平台内容是否属于风险内容，并通过消融实验说明每一类信息对风险识别的贡献。

## 4. 最重要的结论

| 模型组 | F1 | ROC AUC | 应该怎么讲 |
| --- | ---: | ---: | --- |
| `behavior_only` | 0.9729 | 0.9900 | 行为特征在当前数据集上很强，但要谨慎说明可能存在数据分布偏差。 |
| `text_image_behavior` | 0.9640 | 0.9923 | 推荐作为最终三模态工程方案，因为最符合项目主题。 |
| `text_image` | 0.8742 | 0.9416 | 图文融合比纯文本略有提升。 |
| `text_only` | 0.8724 | 0.9366 | 文本语义本身已有稳定识别能力。 |
| `image_only` | 0.6501 | 0.6666 | 图片单独较弱，适合作辅助信号。 |

报告/PPT 主结论建议写：

> 本项目完成了从公开 Weibo 数据、文本特征、图像特征、用户行为特征、多模态融合到消融评估的完整流程。实验表明，完整三模态方案取得 0.9640 的 F1 和 0.9923 的 ROC AUC，能够作为最终工程方案展示；行为特征单独指标最高，但需要在局限性中说明其可能受到数据分布影响。
